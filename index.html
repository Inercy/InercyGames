¿Quiénes somos?

Inercy Games está conformado por tres estudiantes universitarios con una visión de hacer las cosas de la mejor manera posible.

Con esta visión queremos intentar mejorar el aprendizaje en las aulas escolares, apoyándonos de las tecnologías de la información

Herramientas:

Utilizamos tecnología de vanguardia lo cual garantiza un tiempo de vida util mayor para el sistema

  • HTML 5
  • Javascript
  • CSS 3
  • Kinect for Windows

Kinect for Windows

Kinect for Windows es un dispositivo revolucionario que permite percibir y manipular movimiento, imágenes y sonido, todo sin la necesidad de controles.

Tu eres el control

El paquete consta de:

  • 15 juegos educativos compatibles con Kinect for Windows
  • Sistema de estadísticas que monitorea el desempeño de los usuarios
  • Reportes para cada uno de los usuarios que utiliza el sistema
